<h3>Crawling</h3>

- Cocoon에서 크롤링 업무를 맡아 데이터 수집, 정제를 담당하였습니다.
- Selenium과 Beautifulsoup을 이용하였고, 비동기 방식으로 동시에 5개의 사이트를 열어 메타데이터를 크롤링하는 방식으로 코드를 짰습니다.
